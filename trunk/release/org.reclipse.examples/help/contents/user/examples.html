Examples

Blog

With the included Blog example you can see how structural patterns can be detected on a very simple sample application.

To get ready for the example go to File → New → Example... → Blog (Structural Analysis) and finish the wizard.

The newly created project represents the source code and already comes with some resources representing parsed source code.

abstract_factory.gast
An G-AST model parsed by SiSSY that contains a simple structural pattern of the typeAbstract Factory
blog.gast
An G-AST model parsed by SiSSY that contains the source code that is represented by the example project
junit382.gast
An G-AST model parsed by SiSSY that contains the whole JUnit 3.8.2 source
patterns.psc
Some sample structural patterns (model file)
patterns.pscdiagram
Some sample structural patterns (diagram file)

You can start a structural patterns detection over the Reclipse EMF menu and selecting those files.
